# Introduction

Welcome to VitePress documentation.

## What is VitePress?

VitePress is a Static Site Generator (SSG) designed for building fast, content-centric websites.

## Getting Started

```bash
npm install vitepress
```

### Create Your First Page

Create a new markdown file in your docs directory:

```markdown
# Hello VitePress
```

## Features

- Fast build times
- Vue-powered theming
- Markdown extensions

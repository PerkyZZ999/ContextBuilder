# Getting Started

Welcome to our documentation. This guide will help you get started quickly.

## Installation

Install the package using your favorite package manager:

```bash
npm install example-package
```

## Quick Start

Here's a simple example to get you started:

```javascript
import { Example } from 'example-package';

const app = new Example();
app.start();
```

## Configuration

You can configure the package using a configuration file:

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| port | number | 3000 | The port to listen on |
| host | string | localhost | The host to bind to |
| debug | boolean | false | Enable debug mode |

Getting Started

This guide will help you get up and running with VitePress.

Installation

Install VitePress using your favourite package manager:

bun add -d vitepress

File Structure

A typical VitePress project has the following structure:

docs/
├── .vitepress/
│   └── config.ts
├── getting-started.md
└── index.md

TIP

VitePress supports hot module replacement for a fast development experience.

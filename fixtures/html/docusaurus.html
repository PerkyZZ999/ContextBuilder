Installation

Follow these steps to install Example on your system.

Prerequisites

  • Node.js 18 or higher
  • A package manager (npm, yarn, or bun)

Install via npm

npm install example-tool

Configuration

After installation, create a config file:

{
  "name": "my-project",
  "version": "1.0.0"
}
